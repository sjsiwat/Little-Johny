## Priority Issues

**[P1] No prefers-reduced-motion support** — 9+ simultaneous infinite animations with zero @media (prefers-reduced-motion: reduce). Canvas 60fps loop, cat float/blink/tail/ears/whiskers, spin-slow circles, item-in on every list item. Fix: blanket animation suppression + selective restoration. `/impeccable animate`

**[P1] --ink-faint fails contrast on text** — .priority-Low uses #b0a898 (2.24:1 on --surface). WCAG AA requires 4.5:1. Low priority task labels are illegible. Fix: push faint color to ~#7a6e60 (4.5:1) or use --ink-muted for priority text. `/impeccable audit`

**[P1] Numbered 01/02/03 markers violate PRODUCT.md** — quick-start cards use decorative number spans for parallel choices, not a real sequence. Remove qc-num spans; cards work without them. `/impeccable polish`

**[P2] Cream background contradicts "Crafted, not templated"** — --bg: #f5f0e8 is oklch(0.94 0.018 85), solidly in the AI warm-neutral band PRODUCT.md rejects. Shift to near-white with green tint or commit to dark-mode-as-default. `/impeccable bolder`

**[P2] Spring/bounce easing everywhere** — --ease-spring: cubic-bezier(0.34, 1.56, 0.64, 1) on nav hover, card hover, button hover, card-rise, item-in animations. Explicitly banned. Replace with --ease-out (already defined) for interactions. `/impeccable animate`

**[P2] No keyboard shortcuts** — daily-driver tool with zero shortcuts. No view switching, no quick-add, no command palette. `/impeccable harden`

**[P2] No form feedback** — no success/error messages, no delete confirmation, no undo. `/impeccable harden`
